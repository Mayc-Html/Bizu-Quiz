# BIZU QUIZ

Bizu Quiz é um site de quiz interativo criado com o objetivo de tornar o aprendizado mais dinâmico, divertido e acessível. O nome do projeto é uma homenagem ao termo "BIZURADO", expressão usada por nosso antigo professor de informática para se referir a algo dominado, aprendido de forma prática e memorável.

  > SOBRE O PROJETO

O Bizu Quiz nasceu da ideia de unir tecnologia e educação de forma leve e envolvente. Através de perguntas de múltipla escolha, o usuário testa e reforça seus conhecimentos em diversos temas, recebendo feedback imediato sobre seu desempenho.

  > OBJETIVO

Promover a aprendizagem de forma interativa e educativa, incentivando o estudo através da gamificação do conteúdo.

  > LINGUAGENS UTILIZADAS

- HTML — estrutura das páginas
- CSS — estilização e responsividade
- JavaScript — lógica do quiz e interatividade

  > COMO USAR

1. Clone o repositório:

   git clone https://github.com/Mayc-Html/Bizu-Quiz.git
   
2. Acesse a pasta do projeto:
   
   cd Bizu-Quiz
   
3. Abra o arquivo `index.html` no navegador de sua preferência.

Não é necessário instalar dependências ou configurar servidores — o projeto roda direto no navegador.
